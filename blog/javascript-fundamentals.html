Control Flow and Loops

Control flow in JavaScript is the order in which computer executes statements in a script. By default, the code is run from top to bottom, first line to last line. This changes if there are structures such as loops or conditionals which change the control flow.

A loop would interrupt the default control flow and cycle through the loop until there is nothing left to cycle through, and then continue with the rest of the code. A conditional statement would only run if certain conditions are met, and then run code within that conditional statement, before continuing on with the rest of the code.

An analogy of this is an airport. The normal control flow is planes taking off and landing. The control flow is interrupted to cycle through the loops of loading passengers one at a time, loading luggage one at a time and offloading passengers and offloading luggage. Also, if the plane requires fuel, then the fuel is refilled. If the plane needs maintenance, then it is taken out of the control flow until it is ready to fly again.

What Is The DOM?

The DOM is the Document Object Model: a large collection of key: value pairs (aka nodes) that represent all of the structure, content and style within a web page. Because the DOM is represented as an object, programming languages such as JavaScript can interact and change elements of the DOM efficiently.

If you consider a basic HTML document with its myriad elements such as H1, p, form, article, etc., each of these elements is a node. Each node is part of the overall DOM object. Additionally, many of the nodes are objects as well with different properties that can be adjusted. For example, you may have a div with id=”bigList”. What Are Functions?

Functions are basically reusable blocks of code that can be called infinitely throughout a program in an efficient way. They are helpful because if you find yourself using a specific block of code repeatedly, you can instead define this repeating code as a function and then call it whenever it is needed. This helps to cut down on clutter and make your program easier to read and understand.
