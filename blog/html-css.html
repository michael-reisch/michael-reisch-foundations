CSS Concept Research - display: inline; vs display: inline-block;

To properly explain this concept, we must first explore the CSS box model. Basically, the box model consists of a box object in the body of a website. This box object consists of the content, surrounded by padding, surrounded by border, surrounded by margin.

Every html element has a default display property. The display property determines the element layout behaviour in relation to other elements on the page. For example, <div>, <p> and <h1> (among other) elements are block-level elements. This means that the element's box will start on their own line, and take up the entire width of that line. Block-level elements will push other elements onto another line. You can adjust this default behaviour by accessing the display property in CSS like this:

element-selector {display: value}

There are numerous different display values that you can use to manipulate the layout of your web page. The values we will be looking at in this blog post are inline and inline-block.
